A short example page showing how to use headings (<h1>–<h6>), paragraphs, and inline text-formatting tags.

Headings (h1 — h6)

Headings define the structure of your document. Use <h1> for the page title and smaller numbers for sub-sections.

Common inline text-formatting tags

<strong> — indicates strong importance (usually bold).

<em> — emphasizes text (usually italic).

<u> — shows an underline (use sparingly for readability).

<small> — smaller print for fine text.

<mark> — highlights text (like a marker).

<del> — deleted/removed text; <ins> — inserted text.

Use <sub> and <sup> for subscript and superscript: H2O and x2.

Inline code: <code>const x = 10; — use the <code> tag for short code snippets.

Block-level text tags

Blockquote — use <blockquote> for quoted content:

“This is an example blockquote. It’s used to show quoted or cited text.”

Tip: Keep content semantic — use the right tags for meaning, not just appearance. You can style these tags with CSS to change how they look.
