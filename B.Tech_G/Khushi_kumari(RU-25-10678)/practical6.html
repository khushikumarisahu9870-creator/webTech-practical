Inline, Internal & External CSS

Example of all three CSS styles in one webpage

1 Inline CSS

This text uses inline CSS — applied directly inside the HTML tag using the style attribute.

2 Internal CSS

Internal CSS is written inside the <style> tag in the <head> section. It affects all elements that match the defined CSS rules.

3 External CSS

External CSS is stored in a separate file (like style.css) and linked using the <link> tag. It keeps HTML clean and allows styling multiple pages with one stylesheet.
