3. ID Selector (#id)

Used for only one unique element in a page. This blue box has an ID called unique-box.

4. Descendant Selector (parent child)

This rule styles elements that are inside another element. For example, .article p changes the paragraph text color only inside this section.

5. Group Selector (h2, h3)

This heading shares same color as h2

Group selectors style multiple tags together at once, reducing repetition in your CSS.
